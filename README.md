# 🔍 Nexus AI Investigations

> AI-Powered Investigation Management Dashboard built using React, TypeScript & Vite.

---

## 🚀 Overview

Nexus AI Investigations is a modern investigative management platform designed for digital case tracking, officer monitoring, investigation analytics, and annual cost estimation.

The system helps investigation departments:

- Manage digital case records
- Track pending & solved cases
- Monitor investigation statistics
- Calculate investigation expenses
- Maintain officer directories
- Visualize analytics through dashboards

# ⚙️ Installation Guide

## 1️⃣ Clone Repository

```bash
git clone https://github.com/your-username/nexus-ai-investigations.git
```

## 2️⃣ Move into Project

```bash
cd nexus-ai-investigations
```

## 3️⃣ Install Dependencies

```bash
npm install
```

## 4️⃣ Run Development Server

```bash
npm run dev
```
